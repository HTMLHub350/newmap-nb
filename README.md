# New Neverball Map
This is a new neverball map called "Neverball Pixon"

# Where to add the files
1. Download neverball from https://neverball.org
2. Download the files
3. Create new folder called "map-pixon"
4. Add pixo.map and pixo.sol into the "map-pixon" folder
5. Put set-pixon.txt outside of map-pixon (in data/ folder)
6. add "set-pixon.txt" to "sets.txt"

And there you have it, you have Neverball Pixon

## Neverball Pixon
This map is still in development because there is only one level.
